# SECURITY

## Security Posture

NulleCore eAI is a local-first desktop application in Early Preview / Alpha stage.

## Guidance

- Do not upload private models, API tokens, keys, or credentials into shared/public environments.
- Keep sensitive local assets under your own device and access controls.
- Verify release assets against `CHECKSUMS.txt` before installation.

## Reporting Security Concerns

- Report security concerns to: `security@nullecore.local` (placeholder)
- Include steps to reproduce, affected version, and observed behavior.

## Early Preview Notice

- This software is early preview and may contain defects.
- Windows may show an unknown publisher warning until code signing is completed.
