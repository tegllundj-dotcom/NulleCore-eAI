# SECURITY

## Alpha Security Notice

NulleCore-eAI is currently a **Public Alpha Preview**.  
Treat this release as early testing software, not production-grade infrastructure.

## Responsible Disclosure

Please report potential security issues privately before opening a public issue.

- Preferred first contact: open a GitHub issue and mark it clearly as security-sensitive.
- Include: affected version, reproduction steps, expected vs actual behavior, and risk summary.

If a case contains sensitive material, do not post the sensitive material publicly.

## What Not To Post Publicly

Do not post any of the following in public issues/discussions:

- API keys, tokens, credentials, or private model assets
- local machine usernames or private filesystem paths
- raw logs that may include secrets or local environment details
- private workflow files containing personal/business data

## Supported Versions Policy

Security support in this public showcase phase is limited to:

- `v0.1.0-alpha` (latest public alpha release)

Older preview artifacts may not receive fixes or guidance.

## Build Signing Status

- Current installer/build artifacts are **unsigned**
- Windows SmartScreen/unknown publisher warnings may appear
- Verify asset integrity with [CHECKSUMS.txt](CHECKSUMS.txt) before installation
