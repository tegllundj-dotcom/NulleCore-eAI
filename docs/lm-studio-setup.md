# LM Studio Setup

## Purpose

LM Studio is an optional external local dependency for local chat/LLM workflows.

## Basic Steps

1. Install LM Studio locally.
2. Enable local API mode in LM Studio.
3. Launch NulleCore eAI and verify readiness status in the Runtime/Chat surfaces.

## Notes

- NulleCore eAI does not replace LM Studio.
- Model choice and runtime parameters remain user-controlled.
