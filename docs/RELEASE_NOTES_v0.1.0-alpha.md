# NulleCore-eAI v0.1.0-alpha

Release type: **Public Alpha Preview**  
Target platform: **Windows x64**

## What Is Included

- Windows installer package (`.msi`)
- Portable Windows package (`.zip`)
- SHA256 checksum file (`CHECKSUMS.txt`)
- Public documentation for setup, workflows, and troubleshooting
- Curated example workflow files (API-format examples)

## What Is Experimental

- Runtime readiness flow and guidance UX
- Image workflow execution flow and preset governance
- Endpoint-dependent behavior when LM Studio/ComfyUI are unavailable
- Alpha-stage UI and orchestration behavior

## Known Limitations

- Alpha quality: defects and rough edges are expected
- Installer is currently unsigned
- LM Studio and ComfyUI are external dependencies and may require manual setup
- Workflow library is curated but still evolving

## Installer / Signing Status

- Installer: available
- Digital signing: not yet enabled in this alpha release
- Windows may show unknown publisher warnings

## Validation Status

This alpha release was validated as a technical showcase package:

- release assets packaged and uploaded
- checksums generated and published
- release links and documentation structure verified
- public issue templates and testing threads prepared

This is **not** a production readiness certification.

## Technical Evaluation Framing

Use this release for:

- local-first workflow evaluation
- runtime readiness and setup feedback
- installer/packaging feedback
- UI flow and usability feedback for alpha stabilization

Do not use this release as a production deployment baseline.
