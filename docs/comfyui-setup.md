# ComfyUI Setup

## Purpose

ComfyUI is an optional external local dependency for image workflows.

## Basic Steps

1. Install ComfyUI locally.
2. Start ComfyUI and confirm the API endpoint is reachable.
3. In NulleCore eAI, open Image Studio and verify readiness status.

## Notes

- Keep model management local and user-controlled.
- Use curated workflow examples as a starting point.
- This repository does not distribute private models or custom node installers.
