# Local AI Setup

NulleCore eAI is local-first and user-controlled.

## Recommended Setup Flow

1. Install NulleCore eAI.
2. Configure LM Studio if chat/LLM workflows are needed.
3. Configure ComfyUI if image workflows are needed.
4. Confirm runtime readiness from the app Runtime and Image Studio surfaces.

No cloud backend is required for the core local-first usage model.
